Biennale iOS app UI design

iOS app for exhibition visitors who want to get more information via audio guide or text description.
Created with Balsamiq, Sketch and Xcode.

Overview

Simple app for exhibition visitors who want to get more information via audio guide or text description.

Sketches and wireframes

First of all I wrote application definition statement which clearly indetify main purpose of the app. Then sketch my ideas regarding UI on paper based on user cases and functionality requirements. At this stage it’s important to determine navigation style, element’s location, etc. After that I create low-fidelity wireframes in Balsamiq to polish ideas and create deliverables which I could use for presentation or discussions. Those wireframes are without any details and design solutions.

The next step is to create greyscale high-fidelity wireframes in Sketch. At this point it’s important to check that user interface follows best practices and iOS guidelines. Also I create moodboard which helps me to focus on design decisions by gathering desired colors, UI elements, icons at one place.

Design

To begin design process I collect moodboard with different colors and UI elements for inspiration which help me make decisions regarding design. Create style guide and design logo for splash screen, icons and branding.

After that I create design mockups in Sketch based on finished wireframes. Prepare and export assets for different iPhone sizes.

UI development

Then I develop UI in Xcode to provide software developers with storyboard files and image assets.

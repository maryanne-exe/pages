Messenger + task manager concept

Redesigned UI of inner corporate messenger merged with simple task manager.

Overview

The goal was to redesign UI of inner corporate messenger merged with simple task manager.

Messenger

Simple and clean left sidebar for group chats and direct messages.

- Group conversations can be organized for project, topic, team dialogues.
- By clicking plus icon user could join existing chat or create new.
- Only recent dialogues available in a sidebar. Even so, all contacts within the company are in a quick access by clicking plus icon.
- Mainboard chat available for all users. This chat is for general conversations and provides updates about co-workers.
- Search field on the top of the sidebar is for searching chats and messages.

Chat section

- On top of the conversation user have access to related files, to messages which he/she has starred and chat’s members (available for the group chats).
- Also, available search in the conversation.
- Some functions are available on hovering over a message in conversation: view user’s profile, favorite message, forward message.
- By clicking on the user’s name they would be mentioned in the message field.

Task manager

- Task groups in the left sidebar: the user could quickly switch between menu items.
- Also, the user could track tasks related to a specific project.
- Tasks available in the list with short preview. The user could expand task to view more details.
- Priority of the tasks marked by different colors.

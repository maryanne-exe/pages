Eturnity

Eturnity LEA is an intelligent tool for Electric Utilities and Solar Power Installers in Switzerland.

Overview

Eturnity is a Switzerland-based start-up that evaluates the financial feasibility of rooftop solar power and assists Solar Power and Electric Utilities installers.

Complexity of the problem

Solar power systems complexed by comprising of a bunch of the elements: solar panels, batteries, inverters, etc. In addition, the introduction of new power systems, manufacturers, and features makes product extremely data-intense. However, the users care about their clients and businesses looking for the solutions that improve the quality of their work process.

The goal

New interface needs to be flexible and expandable for new functionality, have a clear structure, and emphasize crucial data in the appropriate way.

Work started with learning about the technical side of the field and observations how people keep a record of domestic electricity consumptions and calculate optimal solar system when install one by themselves. In any case — if one is tech savvy and exited to do everything on their own or if one doesn’t want to deal with details — the chance of a mistake is very high; it may be simply not the best system solution.

For every home owner is important to have reliable systems that hospitable and trustworthy.

Refining

With the understanding of the technical side, was introduced the new architecture for the main screens throughout the project.

New navigation

New features required creating a brand new navigation system. Distinct, Visual, and Scalable.

Library

The introduction of the Components Library allows the user to store systems components for the future usage. The consistent interface allows easily browse the components and change settings.

Dashboard

New navigation system allows implementing accessible project-level overview. Crucial data and important updates on the platter.

New systems

One of the challenges of the navigation redesign was to implement two new systems - Heat Pumps and E-Mobility - besides the original Photovoltaic system. The user has to have a clear understanding of their position in the structure levels. In addition, to meet marketing goal to provide the user with a clue of a possible extension for these systems.

UI system

To cut down on development time, was built an extensive style guide with reusable patterns.

Mobile solution

New design is responsible and supports mobile and tablet views.

Feedback and errors

No doubts the user should have support in difficult situations.

404 and report a problem pages
Onboarding gif animation, html prototype
Errors handling

Reflection

At the end, Eturnity acquired an architecture, design, and brand that embodied initially set attributes: scalable, clean, and supportive. That is the strong basis that allows the project to evolve and deliver better user experience.
